# naruto
Projeto utilizando Spring Boot, MySQL e Thymeleaf.

## Dependências
- JDK: 8
- Apache Maven: 3.6.3
- Spring Boot: 2.4.0
- Thymeleaf: 3.0.11
- Bootstrap: 4.6.2
- MySQL: 8.0.22

## Instalação
Baixando as dependências

```bash
  cd naruto
  mvn install
```

## Acessando
- http://localhost:8080/ninjas
- http://localhost:8080/api/ninjas
